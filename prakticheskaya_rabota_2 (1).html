Базовые трансформации — rotate и scale
transform: rotate(), scale()
Начать ▶
Задание
Создайте HTML-файл с тремя одинаковыми квадратами. К первому не применяйте ничего, второй поверните на 30 градусов, третий увеличьте в 1.4 раза.
Ожидаемый результат
HTML — структура страницы
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Трансформации</title>
    <style>
      /* сюда пишем CSS */
    </style>
  </head>
  <body>
    <div></div>
    <div class="rotate"></div>
    <div class="scale"></div>
  </body>
</html>
CSS — стили
div {
  width: 80px;
  height: 80px;
  background: #ccc;
  border: 1px solid #333;
  display: inline-block;
  margin: 30px;
}
.rotate { transform: rotate(30deg); }
.scale  { transform: scale(1.4); }
Подсказка: угол в rotate() задаётся в градусах — deg. Число в scale() — это множитель размера: 1 — без изменений, 0.5 — в два раза меньше, 2 — в два раза больше.

translate, skew и transform-origin
смещение, наклон и точка опоры
Заблокировано ▶
Задание
Создайте три квадрата разного цвета:
  • первый сдвиньте на 20px вправо и 10px вверх (translate);
  • второй наклоните на -20 градусов по оси X (skew);
  • третий поверните на -30 градусов вокруг его левого верхнего угла (transform-origin).
Ожидаемый результат
HTML
<div class="move"></div>
<div class="skew"></div>
<div class="origin"></div>
CSS
div {
  width: 80px;
  height: 80px;
  border: 1px solid #333;
  display: inline-block;
  margin: 30px;
}
.move {
  background: #b5d0ff;
  transform: translate(20px, -10px);
}
.skew {
  background: #fcb5b5;
  transform: skew(-20deg, 0);
}
.origin {
  background: #c3f5c3;
  transform-origin: left top;
  transform: rotate(-30deg);
}
Запомните: transform-origin меняет точку опоры для трансформации. Без него поворот идёт вокруг центра элемента. Значения: center, left top, right bottom, left bottom, right top.

Плавные переходы — transition
плавное изменение свойств при :hover
Заблокировано ▶
Задание
Создайте два квадрата. При наведении мыши оба должны становиться красными и чуть больше, но:
  • у первого переход происходит мгновенно (без transition);
  • у второго — плавно за 1 секунду.
Наведите курсор в превью ниже — убедитесь, что разница видна.
Ожидаемый результат (наведите мышь)
instant
smooth
HTML
<div class="instant">instant</div>
<div class="smooth">smooth</div>
CSS
div {
  width: 90px;
  height: 90px;
  background: #ccc;
  border: 1px solid #333;
  margin: 30px;
  display: inline-block;
}
.smooth {
  transition: background 1s, transform 1s;
}
div:hover {
  background: #f87171;
  transform: scale(1.15);
}
Краткая форма transition
/* transition: свойство длительность функция задержка */
transition: background 1s ease-in-out 200ms;

/* чтобы плавно менялись ВСЕ свойства сразу: */
transition: all 1s;
Важно: transition прописывается в обычном состоянии элемента, а не в :hover. Иначе плавным будет только переход «туда», но не «обратно».
4
Анимации — @keyframes
ключевые кадры, бесконечное воспроизведение
Заблокировано ▶
Задание
Создайте квадрат, который сам (без наведения мыши) плавно перебирает цвета: красный → жёлтый → зелёный → синий → фиолетовый. Анимация должна идти бесконечно.
Ожидаемый результат
квадрат должен сам менять цвет — смотрите превью несколько секунд
HTML
<div></div>
CSS
@keyframes colorAnimation {
  0%   { background: red; }
  25%  { background: yellow; }
  50%  { background: green; }
  75%  { background: blue; }
  100% { background: violet; }
}
div {
  width: 90px;
  height: 90px;
  border: 1px solid #333;
  animation-name: colorAnimation;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}
Краткая форма animation
/* animation: имя длительность функция повторы направление задержка заливка */
animation: colorAnimation 4s ease-in-out infinite alternate;
Полезно знать:
  • infinite — бесконечное повторение;
  • alternate — чередование «туда-обратно»;
  • forwards — остановиться на последнем кадре;
  • from и to — то же самое, что 0% и 100%.

Самостоятельное задание
две фигуры + анимированный баннер
Заблокировано ▶
Задание
Создайте страницу, на которой:
  • есть две фигуры — одна обычная, вторая как-либо трансформирована (rotate, scale, skew — на ваше усмотрение);
  • у трансформированной фигуры должен быть плавный переход (transition) — например, при наведении мыши она крутится, меняет цвет, размер и т.д.;
  • на странице должен быть баннер с анимацией (@keyframes) — меняющийся фон, появляющийся текст, бегущая надпись — что угодно.
Можно сделать гротескно или красиво — главное, чтобы все три элемента были на странице.
Примерный вид результата (наведите мышь на розовый квадрат)
Только в этом месяце
Скидки до 20%
Подсказка — каркас CSS для баннера
@keyframes banner {
  10%  { background: #008978; }
  40%  { background: #34495e; }
  80%  { background: green; }
}
@keyframes text1 {
  10% { opacity: 1; }
  40% { opacity: 0; }
}
@keyframes text2 {
  30% { opacity: 0; }
  60% { opacity: 1; }
}

.banner {
  width: 600px;
  height: 120px;
  position: relative;
  animation: banner 6s infinite;
}
.text1, .text2 {
  position: absolute;
  width: 100%;
  line-height: 120px;
  text-align: center;
  color: #fff;
  font-size: 40px;
  opacity: 0;
}
.text1 { animation: text1 6s infinite; }
.text2 { animation: text2 6s infinite; }
Подсказка — HTML структура баннера
<div class="banner">
  <div class="text1">Только в этом месяце</div>
  <div class="text2">Скидки по 20%</div>
</div>
Важно: на странице должны быть все три элемента — обычная фигура, трансформированная фигура (с плавным переходом), анимированный баннер.
