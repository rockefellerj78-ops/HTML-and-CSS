Создайте HTML-файл с четырьмя блоками: шапка, сайдбар, основное содержимое, футер. Все блоки расположите друг под другом.

HTML — структура страницы
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Блочная вёрстка</title>
    <style>
      /* сюда пишем CSS */
    </style>
  </head>
  <body>
    <div id="header">Шапка сайта</div>
    <div id="sidebar">Сайдбар</div>
    <div id="main">Основное содержимое</div>
    <div id="footer">Футер</div>
  </body>
</html>
CSS — стили
div {
  margin: 10px;
  border: 1px solid black;
  font-size: 20px;
  height: 80px;
}
#header  { background-color: #ccc; }
#sidebar { background-color: #ddd; }
#main    { background-color: #eee; height: 200px; }
#footer  { background-color: #ccc; }
Подсказка: CSS пишите внутри тега <style> в <head>. Сохраните файл как index.html и откройте в браузере.

Задание

Измените CSS для #sidebar и #main
#sidebar {
  background-color: #ddd;
  float: left;
  width: 150px;
}
#main {
  background-color: #eee;
  height: 200px;
  margin-left: 170px; /* 150px + 10px + 10px */
}
Подсказка: margin-left у #main должен быть больше ширины сайдбара — иначе блоки наедут друг на друга.

Самостоятельное задание
Создайте страницу сайта с реальным содержимым
Заблокировано ▶
Задание
Создайте страницу сайта. Требования:
  • Шапка с названием сайта
  • Левое меню с 4 ссылками: Главная, Блог, Контакты, О сайте
  • Основной блок с заголовком <h2> и текстом <p>
  • Футер с текстом «Copyright © 2026»
  • У body, h2, p убрать все отступы
Подсказка — CSS
body, h2, p { margin: 0; padding: 0; }
body        { font-size: 18px; }

#header  { background: #eee; border-bottom: 1px solid #ccc; height: 80px; }
#wrapper { background: #ddd; }

#menu {
  float: left;
  width: 150px;
}
#main {
  background: #f7f7f7;
  border-left: 1px solid #ccc;
  margin-left: 150px;
  padding: 10px;
}
#footer { border-top: 1px solid #ccc; background: #dedede; }
Подсказка — HTML структура
<div id="header">Название сайта</div>
<div id="wrapper">
  <div id="menu">
    <ul>
      <li><a href="#">Главная</a></li>
      <li><a href="#">Блог</a></li>
      <li><a href="#">Контакты</a></li>
      <li><a href="#">О сайте</a></li>
    </ul>
  </div>
  <div id="main">
    <h2>Заголовок страницы</h2>
    <p>Текст содержимого...</p>
  </div>
</div>
<div id="footer">Copyright © 2026</div>
Обратите внимание: меню и #main оба находятся внутри блока #wrapper — это важно для правильной работы float.
