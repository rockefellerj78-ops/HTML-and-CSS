Цель работы

Закрепить базовую структуру HTML-документа, повторить назначение <head> и <body>, а также научиться добавлять новые смысловые блоки и оформлять их с помощью CSS.

Структура страницы

  • header - шапка сайта (название, подзаголовок)
  • nav - меню (ссылки на разделы)
  • main - основное содержимое
  • footer - подвал (контакты/автор/год)

Сейчас это один файл. В следующих лабораторных сюда легко добавятся новые страницы (about.html, gallery.html и т.д.).

Какие теги использованы

  1. <!doctype html> - объявление HTML5
  2. <html lang="ru"> - язык документа
  3. <meta charset="utf-8"> - кодировка
  4. <meta name="viewport" ...> - корректная адаптивность
  5. <title> - заголовок вкладки браузера
  6. <link rel="stylesheet"> - подключение CSS
  7. <header> <nav> <main> <section> <footer> - семантические блоки

Специальные символы HTML

Некоторые символы в HTML записываются не напрямую, а через специальные последовательности.

  • &lt; = <
  • &gt; = >
  • &amp; = &
  • &copy; = ©
  • &quot; = "
  • &nbsp; = неразрывный пробел

Пример вывода тега как текста: &lt;h1&gt;Заголовок&lt;/h1&gt;

Пример с амперсандом: A & B.

Теги <code> и <pre>

<code> обычно используют для коротких фрагментов кода прямо в тексте, а <pre> — когда нужно сохранить переносы строк и пробелы.
